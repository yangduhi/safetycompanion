## 해석 규칙
- 단계 순서는 `plan.md`를 따른다.
- 경로, 환경, 성공 기준, 재실행 규칙은 `spec.md`를 따른다.
- 체크리스트는 `tasks.md`를 따른다.
- step 문서 내 산출물 경로가 `outputs/...` 계열일 경우, 실제 실행 산출물은 `outputs/<run_id>/...`에 저장한다.
