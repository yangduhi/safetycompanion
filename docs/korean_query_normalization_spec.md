# Korean Query Normalization Spec

## 목적
한국어 질의와 영어 원문 사이 lexical bridge를 강화한다.

## 주요 기능
- spaced acronym collapse
- route-aware lexical expansion
- domain lexicon expansion
- bilingual search query 생성

## 예시
- `관련 과정` -> `course seminar training`
- `페이지` -> `page where found`
- `약어 뜻` -> `abbreviation stands for`
- `충돌시험` -> `crash test`
- `수동안전` -> `passive safety`
- `능동안전` -> `active safety`
- `더미` -> `dummy atd`
