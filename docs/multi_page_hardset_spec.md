# Multi Page Hardset Spec

## 목적
multi-page 질의를 single-entry 문제로 축소하지 않도록 별도 hard set을 유지한다.

## 우선 포함 질문 유형
- THOR 관련 페이지 모음
- dummy landscape 관련 페이지
- ATD / HIII / SID-IIs 혼합 질의
- 관련 페이지 정리 / 모여 있는 페이지 / 어디에 있나
- 한국어 multi-page paraphrase

## 필수 메타데이터
- `question`
- `difficulty`
- `question_type`
- `expected_pdf_pages`
- `expected_titles`
- `expected_grouping_hint`
